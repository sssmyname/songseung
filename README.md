# 🎵 My Music Rater (자작곡 재생 및 별점 평가 시스템)

## 1. Title & Abstract
- **Title:** My Music Rater
- **Abstract:** 사용자가 직접 제작한 여러 개의 음악 파일을 웹 환경에서 손쉽게 선택하여 감상할 수 있는 대시보드입니다. Streamlit을 활용하여 구축되었으며, 청취 직후 1~5점 척도로 곡에 대한 직관적인 별점 평가를 남길 수 있는 상호작용형 UI를 제공합니다.

## 2. Key Features
- **오디오 자동 스캔 및 재생:** 폴더 내의 `.mp3`, `.wav` 파일 목록을 자동으로 불러와 드롭다운으로 선택하고 재생할 수 있습니다.
- **사용자 평가 시스템 (Rating):** 곡을 감상한 후 1점부터 5점까지 슬라이더를 통해 즉각적인 피드백을 남길 수 있습니다.
- **직관적인 UI/UX:** 복잡한 세팅 없이 깔끔하고 반응형인 컨트롤 패널을 제공합니다.

## 3. Getting Started
이 프로젝트를 로컬 환경에서 실행하려면 아래의 순서를 따라주세요.

### 요구 사항
- Python 3.8 이상
- Streamlit 라이브러리 설치 필요 (`pip install streamlit`)

### 실행 방법
1. 본 레포지토리를 클론(Clone)하고 해당 폴더로 이동합니다.
2. 터미널에 아래 명령어를 입력하여 앱을 실행합니다.
   ```bash
   python3 -m streamlit run app.py
